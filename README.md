# ✨ VogueWear

A premium fashion ecommerce experience built with React and Vite, designed with a strong focus on aesthetics, user experience and modern shopping interactions.

## Live Demo

🔗 https://vogue-wear-fawn.vercel.app

## GitHub Repository

🔗 https://github.com/aakkanksha122/VogueWear

## Features

* Premium editorial inspired UI
* Product listing and category filtering
* Product Details Page (PDP)
* Wishlist functionality
* Shopping cart management
* Search experience
* New Arrivals collection
* Sale collection
* Trending products carousel
* AI-powered Style Concierge recommendations
* Responsive design across screen sizes

## AI Style Concierge

The AI Stylist feature helps customers discover products based on occasions and styling preferences.

Examples include:

* Office Wear
* Brunch Looks
* Wedding Guest Outfits
* Vacation Styling

Based on the selected occasion, VogueWear recommends curated products from the catalog using tag-based recommendations.
